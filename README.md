# Projeção de Embeddings - SRI

## Descrição

Este repositório contém os notebooks e os arquivos gerados para a atividade de Projeção de Embeddings da disciplina de Sistemas de Recuperação da Informação (SRI).

O objetivo da atividade foi gerar representações vetoriais (embeddings) de documentos, tokens e sentenças utilizando os notebooks disponibilizados pelo professor e visualizar os resultados por meio do TensorFlow Embedding Projector.

## Corpus Utilizado

Foi utilizado um corpus próprio contendo 30 documentos em português relacionados aos temas:

* Inteligência Artificial
* Ciência de Dados
* Python
* SQL
* Machine Learning
* Automação de Processos
* Processamento de Linguagem Natural (PLN)
* Governança de Dados
